GOAL
Build the full portfolio website from the provided markdown content and project instructions.

CONTEXT
This repository contains:
- copilot-instructions.md (rules and structure)
- markdown content for index, projects, cv, and project pages

This is a minimal static website (no frameworks).

TASK
1. Create all required HTML files:
   - index.html
   - projects.html
   - cv.html
   - contact.html
   - /projects/flyway-model.html
   - /projects/arctic-climate.html

2. Convert the markdown content into clean semantic HTML

3. Ensure all pages:
   - use the same navigation
   - link to each other correctly
   - follow the defined structure

4. Link the CSS file (/assets/css/style.css) in all pages

5. Keep layout consistent across all pages

CONSTRAINTS
- No frameworks (no React, no Tailwind, no Bootstrap)
- Use semantic HTML (header, nav, main, section, footer)
- Keep structure clean and readable
- Do not overengineer
- Follow copilot-instructions.md strictly

OUTPUT
- Fully working static website

- Consistent structure across pages
